# BoweFlex

This is meant to (eventually) be the site for all things.. me. I'd like to get this to the point of being a semi-frequent "blog", as well as a portfolio of either links to github repos or other forms of projects.

However, there is a good bit of building that needs to be done before that can happen. My main goal at this time is to get some experience building a site - more specifically, I've been hearing a lot about the Go + htmx stack and have enjoyed my little bit of work with Go so far so I want to try it out here.

I think this is a really good opportunity for me to start with that, because 1. while I understand most of the basics of hosting a site, this allows me to avoid that and focus on development, and 2. I don't have to purchase and maintain a domain.
